# Currency Converter

This is a simple currency converter web app built as a JavaScript practice project. It uses HTML, CSS, and vanilla JavaScript to demonstrate currency selection, conversion logic, and a clean responsive UI.

## Purpose

- Practice JavaScript fundamentals in a small project
- Learn DOM manipulation, event handling, and API integration
- Prepare for a future migration to Angular for a more advanced frontend implementation

## Features

- Select source and target currencies
- Enter an amount to convert
- Display converted result
- Uses currency codes and country data for dropdown management

## Files

- `index.html` — main page markup
- `style.css` — styles for layout and design
- `script.js` — conversion logic and app behavior
- `country-code.js` — currency code and country mapping data

## How to use

1. Open `index.html` in a browser.
2. Enter an amount.
3. Choose the source currency.
4. Choose the target currency.
5. Click the button to see the converted result.

## Notes

- This repo is intentionally kept simple as a JavaScript practice repository.
- The goal is to use this project as a foundation before learning and building the same app in Angular.
